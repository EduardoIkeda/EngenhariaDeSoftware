      Main Description
      Desenvolver o sistema responsável pela manipulação de dados, regras de negócio, segurança, e comunicação com o banco de dados e o front end. Esta camada atua como o cérebro do sistema.
      Steps
      Definir a estrutura do projeto de back end (ex.: arquitetura REST).
      Configurar a comunicação com o banco de dados.
      Implementar os modelos e controladores principais.
      Criar endpoints para manipulação de dados (parcialmente integrado à API).
      Realizar testes de unidade para cada componente desenvolvido.
