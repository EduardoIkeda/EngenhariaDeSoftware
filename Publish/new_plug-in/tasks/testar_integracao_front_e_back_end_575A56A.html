Task: Testar integracao front e back end
Garantir que o front end e o back end se comuniquem corretamente através da API.
Main Description
Testar o sistema completo para verificar se as interfaces de usuário (front end) estão funcionando corretamente com a lógica de negócio (back end), garantindo uma comunicação adequada e sem falhas.
Steps
Configurar o front end para realizar chamadas à API.
Testar se as requisições do front end retornam os dados esperados da API.
Verificar se as respostas da API são corretamente exibidas no front end (ex.: listas, detalhes).
Testar a manipulação de dados pelo front end com base nas respostas do back end.
Realizar testes de usabilidade para garantir que o fluxo de dados está correto do ponto de vista do usuário final.
