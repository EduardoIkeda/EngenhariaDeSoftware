Task: Integrar front end e back end
Garantir a comunicação correta entre o front end e o back end através da API.
Main Description
Esta tarefa envolve conectar a interface de usuário desenvolvida no front end com a API do sistema, garantindo que as requisições enviadas pelo front end sejam corretamente processadas pelo back end e que as respostas sejam exibidas adequadamente na interface.
Steps
Configurar o cliente HTTP no front end (ex.: Axios, Fetch).
Criar funções para chamar os endpoints da API.
Integrar as chamadas da API aos componentes de interface.
Tratar erros e estados de carregamento.
Testar o fluxo completo de dados entre o front end e o back end.
