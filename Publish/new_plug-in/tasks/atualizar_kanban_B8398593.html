Task: Atualizar kanban
Manter o quadro Kanban atualizado com as tarefas e status de cada atividade.

      Main Description
      Esta tarefa envolve a atualização contínua do quadro Kanban, refletindo o progresso das tarefas e garantindo que a equipe tenha uma visão clara do fluxo de trabalho. O responsável deve mover as tarefas pelas colunas apropriadas (por exemplo, "A fazer", "Em progresso", "Concluído") e ajustar prazos ou prioridades quando necessário. É fundamental para garantir que o time esteja alinhado com os status das entregas e prazos.
      Steps
      Revisar o status atual das tarefas no quadro Kanban.
      Mover as tarefas para as colunas adequadas com base no progresso.
      Adicionar novas tarefas ou remover tarefas concluídas.
      Atualizar prazos e prioridades conforme necessário.
      Revisar o quadro com a equipe durante as reuniões de acompanhamento.
